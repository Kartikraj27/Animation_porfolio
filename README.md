# Cinematic Portfolio

Premium cinematic portfolio with Three.js bokeh particles, GSAP animations, and video hero.

## Setup

1. Add your video file as `public/intro.mp4`
2. Edit your name in `components/HeroSection.tsx` (line with YOUR / NAME)
3. Update email in `components/ContactSection.tsx`
4. Update projects in `components/ProjectsSection.tsx`

## Deploy to Vercel

1. Push this repo to GitHub
2. Import on vercel.com — it auto-detects Next.js
3. Deploy ✅

## Local Dev

```bash
npm install
npm run dev
```
